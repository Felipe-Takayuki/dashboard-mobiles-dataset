# Dashboard Mobile

## Como Instalar ?

- No linux
``` 
git clone https://github.com/Felipe-Takayuki/dashboard-mobiles-dataset.git 

cd dashboard-mobiles-dataset

python3 -m venv venv 
source venv/bin/activate
pip install -r requirements.txt
```
- No Windows 
``` 
git clone https://github.com/Felipe-Takayuki/dashboard-mobiles-dataset.git 

cd dashboard-mobiles-dataset

python -m venv venv 
venv\Scripts\activate.bat 
pip install -r requirements.txt

```
